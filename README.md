# Simple PHP CRUD Project

This is a simple practice project developed with pure PHP to demonstrate the basic CRUD operations and user authentication.

## Features

User Registration (Create)

User Login

User Profile Update (Read & Update)

User Logout

Basic CRUD operations

Simple authentication system using PHP sessions

## Project Purpose

The main goal of this project is to practice:

PHP fundamentals

Working with forms and POST requests

CRUD operations

User authentication (register, login, logout)

Updating user profile data

Session handling

## Technologies Used

PHP (Core PHP, no framework)

MySQL (for database)

HTML / CSS (basic)

phpMyAdmin (optional, for database management)

## How It Works

A user can register with basic information.

After registration, the user can log in.

Once logged in, the user can view and update their profile.

The user can log out, which destroys the session.
